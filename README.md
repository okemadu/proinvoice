# ProInvoice
A clean, professional, and mobile-friendly receipt generator for independent business owners and service technicians. 

ProInvoice allows you to quickly generate official-looking receipts on the go. Designed with a minimalist aesthetic, it ensures your clients get a professional document for their records instantly.

## Features
* **Professional Layout:** Generates clean, printer-friendly receipts with a classic business design.
* **Multi-Currency Support:** Includes support for USD, MWK, ZMW, NGN, and more.
* **Flexible Payments:** Integrated support for sharing direct payment links (Grey.co, etc.) or bank transfer details.
* **Mobile Optimized:** Built to work seamlessly on smartphones, perfect for site visits and fieldwork.
* **No Bloat:** A lightweight, ad-free tool that loads instantly.

## How to Use
1. **Input Details:** Enter your business name, client details, description of service, and total amount.
2. **Set Payment Info:** Add your Grey.co link or bank account details for easy payment.
3. **Generate:** Click "GENERATE OFFICIAL RECEIPT."
4. **Print/Save:** The app will open a clean receipt in a new tab, which you can then print or save as a PDF.

## Deployment
This project is built using standard HTML, CSS, and JavaScript. It is optimized to be hosted on **GitHub Pages**.

To deploy your own version:
1. Fork this repository.
2. Go to **Settings > Pages**.
3. Set the source branch to `main` (or `master`).
4. Your app will be live at `https://yourusername.github.io/proinvoice/`.
